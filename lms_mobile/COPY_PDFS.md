# Hướng dẫn copy file PDF

Để màn hình Chương trình học hoạt động, bạn cần copy các file PDF vào thư mục `assets/pdfs/`:

1. Copy file `Chương trình đào tạo ngành Công nghệ thông tin.pdf.pdf` từ thư mục gốc vào `lms_mobile/assets/pdfs/cntt.pdf`
2. Copy file `Chương trình đào tạo ngành ĐT-VT.pdf.pdf` từ thư mục gốc vào `lms_mobile/assets/pdfs/dtvt.pdf`

Sau khi copy xong, chạy:
```bash
flutter pub get
```

Để cài đặt package `pdfx` để hiển thị PDF.
